# Projeto com React-Native
Esse projeto é protótipo que não esta em pleno funcionamento, desenvolvido em parceria com a empresa TAIFF, envolvendo aplicação mobile, API backend e integração com hardware para testes automatizados de secadores de cabelo.

## Descrição do projeto:

Desenvolvimento de aplicação mobile em React Native (Expo) para leitura de QR Code dos modelos de secadores, permitindo ao usuário acessar informações e resultados de testes. Esse projeto estava em construção em paralelo ao desenvolvimento de uma bancada de testes de secadores utilizando um arduino para controle de motores de passo e pneumaticos, conectados a uma API REST em Java com documentação dos endpoints utilizando Swagger, responsável pelo envio e recebimento de dados de testes para visualização.

Minha participação desse projeto foi fazer a ponte entre as tecnologias, onde tive o privilégio de construir o código em arduino para bancada de teste com sensores de temperatura, API REST em Java para conectar ao frontend e com a bancada de testes. O projeto como um todo não foi finalizado, mas tivemos diversos aprendizados para a vida profissional.
